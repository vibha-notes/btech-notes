Express JS
- templating engines like Pug, Mustache, EJS
- converted to HTML
- Model-View-Controller (MVC): data(express)-frontend(react)-server(nodejs)


Install express
- npm init
- npm install -g express (globally)
- npm install express --save (project folder)


MVC Separations of Concerns:
Model - data (REST APIs - ExpressJS)
View - ReactJS
Controller - intermediate server (NodeJS)
