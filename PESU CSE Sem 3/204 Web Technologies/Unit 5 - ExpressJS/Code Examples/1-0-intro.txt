API 
- Application Program Interface
- Provides some function/service
- returnvalue getmovies(arguments) -> API 
- boolean savereview(movieid, reviewcomments)
- API name is a verb

- Platform dependent (language, os)
- request/response (client/server)
- text response - html/json/xml


API on web

REST APIs
- REST: simpler, shortcomings of SOAP
- REpresentational State Transfer (resource oriented architecture)
- Separation of concerns - client/server
- stateless
- cacheable
- uniform interface
- layered architecture
- code on demand

HTTP Methods - CRUD
- GET: read-list or read obj
- POST: create
- PUT: update
- DELETE: delete
